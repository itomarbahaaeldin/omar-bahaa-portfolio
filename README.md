## 📋 Overview

My personal portfolio website — a dark-themed, single-page application showcasing my projects, skills, and background as a Full-Stack Engineer and ML practitioner. Built entirely with vanilla HTML, CSS, and JavaScript; no frameworks, no build tools, zero dependencies.

<br/>

## ✨ Design Highlights

- **Dark aesthetic** with a custom `#080c12` base and `#00d4ff` / `#0066ff` / `#7b2fff` accent palette
- **Animated grid background** using CSS pseudo-elements
- **Glassmorphism nav** with `backdrop-filter: blur(20px)`
- **Bebas Neue** display font paired with **DM Sans** for body text
- **JetBrains Mono** for code-style elements
- **Scroll-triggered animations** and staggered entrance reveals
- Fully responsive — mobile, tablet, desktop

<br/>

## 🗂️ Sections

| Section | Content |
|---------|---------|
| Hero | Name, role, CTA buttons, animated intro |
| About | Background, education, personal bio |
| Skills | Tech stack grouped by category with icons |
| Projects | Featured project cards with links and tech tags |
| Contact | Email, LinkedIn, GitHub links |

## 🚀 Run Locally

No build step required — just open the file:

```bash
git clone https://github.com/itomarbahaaeldin/omar-bahaa-portfolio.git
cd omar-bahaa-portfolio
open index.html        # Mac
start index.html       # Windows
xdg-open index.html    # Linux
```

Or serve it:
```bash
npx serve .
# → http://localhost:3000
```

## 🚢 Deployment

Hosted for free on **GitHub Pages**. To deploy your own fork:

1. Fork the repo
2. Go to Settings → Pages
3. Set Source to `main` branch → `/ (root)`
4. Your site is live at `https://yourusername.github.io/omar-bahaa-portfolio/`
